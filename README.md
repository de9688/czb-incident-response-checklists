# CZB Incident Response Checklists

Public checklists and templates for crypto incident response, wallet compromise triage, phishing approval response, and evidence-safe communication.

## Use Cases

- Suspected private-key or seed phrase exposure
- Malicious approval or phishing signature incident
- Stolen-asset tracing preparation
- Internal incident handoff between security, legal, and operations teams
- Public communication planning after a confirmed event
